HTML Structure

Traveltrek template is based on Latest Bootstrap 4 Framework. Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases.

Here is an example about bootstrap grid. If you need more information about bootstrap grid, please visit this site: http://getbootstrap.com

It's possible to change the JS settings by extending the defaults from the template, that way is so much easier to update the theme changing only one JS file. all plugins activation files is here. open "Traveltrek/js/active.js" file. Here are a few samples on how to do that:

1. Changing the slider loop, autoplay and speed option.

			$(".trips-slider").owlCarousel({
				loop:true,
				autoplay:true,
				smartSpeed: 500,
			});

2. Changing The counterup timing.

			$('.number').counterUp({
				time: 1000
			});

3. Changing the "video popup" delay.

			$('.video-popup').magnificPopup({
				removalDelay: 300,
			});

How can i use google map location?

Open "Traveltrek/js/map-script.js" file. Then find the google map comments tag. Just replace your own google map latitude and longitude address. See the red shape below to replace your google map latitude and longitude address.

How can i add email address into contact form?

Open "traveltrek/mail/mail.php" file. Just replace your own email address for receiving email from contact form.

How can i replace youtube video background?

Open "traveltrek/index3.html" file. Just replace your own youtube video link. see the red shape bellow to replace video background.
